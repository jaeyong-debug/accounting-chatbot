📚 지식베이스 업데이트

시스템 프롬프트를 직접 편집하고 Worker에 저장하면 즉시 반영됩니다. (재배포 불필요)

🚀 Cloudflare Worker 배포 가이드
1단계 — Cloudflare 계정 생성
  → https://dash.cloudflare.com/sign-up (무료)

2단계 — Node.js 설치 (없는 경우)
  → https://nodejs.org → LTS 버전 다운로드

3단계 — Wrangler 설치 및 로그인
  터미널에서: 
npm install -g wrangler
wrangler login
4단계 — KV 네임스페이스 생성
wrangler kv namespace create "USAGE_KV"
→ 출력된 id를 wrangler.toml의 id 부분에 입력

5단계 — 시크릿 설정
wrangler secret put CLAUDE_API_KEY
# → sk-ant-api03-... 입력

wrangler secret put ADMIN_PASSWORD
# → admin1234 입력
6단계 — 배포
wrangler deploy
→ 출력된 URL (예: https://accounting-chatbot.xxx.workers.dev) 을
   챗봇 페이지와 이 관리자 페이지의 Worker URL 필드에 입력

✅ 완료! 이후 사용자들은 API 키 없이 챗봇을 사용할 수 있습니다.
